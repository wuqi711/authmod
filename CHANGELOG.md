# 更新日志

本文档记录AuthMod的所有重要更改。

## [1.0.0] - 2024-10-30

### 新增功能
- ✨ 玩家注册系统 (`/register` 命令)
- ✨ 玩家登录系统 (`/login` 命令)
- ✨ 密码修改功能 (`/changepassword` 命令)
- ✨ SHA-256密码加密（带随机盐值）
- ✨ SQLite数据库存储玩家数据
- ✨ 会话管理系统（断线重连自动登录）
- ✨ 登录超时自动踢出功能
- ✨ 登录失败次数限制
- ✨ 管理员命令系统：
  - `/authmod unregister` - 注销玩家账户
  - `/authmod resetpassword` - 重置玩家密码
  - `/authmod kickunlogged` - 踢出未登录玩家
  - `/authmod reload` - 重载配置
- ✨ 可配置的服务器设置
- ✨ 中英文语言支持

### 安全功能
- 🔒 限制未登录玩家移动
- 🔒 限制未登录玩家聊天（除命令外）
- 🔒 限制未登录玩家交互方块
- 🔒 限制未登录玩家攻击实体
- 🔒 保护未登录玩家免受伤害
- 🔒 限制未登录玩家破坏方块

### 配置选项
- ⚙️ 登录超时时间
- ⚙️ 会话超时时间
- ⚙️ 最小密码长度
- ⚙️ 会话管理开关
- ⚙️ 登录传送开关
- ⚙️ 保护半径设置
- ⚙️ 最大登录尝试次数

### 技术细节
- 📦 基于Forge 1.20.1 (47.2.0)
- 📦 使用SQLite JDBC 3.43.0.0
- 📦 Java 17兼容
- 📦 支持服务器端运行
